---
title: Debian Security Repo Mirrors Usage Guide
cname: 'debian-security'
---

In general, replace the default Debian source address `http://security.debian.org/` with the mirror address in the `/etc/apt/sources.list` file.

```properties
# By default, source code mirrors are commented out to improve apt update speed. Uncomment them if needed.
{{is_sid}}deb {{http_protocol}}{{mirror}} {{release_name}}{{security}} main contrib non-free{{has_nfw}}
{{is_sid}}{{enable_source}}deb-src {{http_protocol}}{{mirror}} {{release_name}}{{security}} main contrib non-free{{has_nfw}}
```
